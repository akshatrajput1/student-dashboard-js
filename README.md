# 📊 Student Dashboard (JavaScript)

This project demonstrates core JavaScript array methods:

- map()
- filter()
- reduce()
- forEach()
- destructuring

## 🚀 Features

- Show all students
- Filter toppers (>80 marks)
- Calculate average marks
- Group students by subject

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (ES6)

## 🎯 Purpose

This project is built for practicing JavaScript fundamentals before starting React.

## 📸 Output

Simple UI with buttons to interact with student data.
